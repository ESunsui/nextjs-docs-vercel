# 安装VMT

首先，下载VMT的安装包。[vmt_013c](https://github.com/gpsnmeajp/VirtualMotionTracker/releases/download/v0.13c/vmt_013c.zip)

解压后，您会看到一系列文件。

<center>![install_vmt_1](image/install_vmt/1.jpg)</center>

打开vmt_manager文件夹。

<center>![install_vmt_2](image/install_vmt/2.jpg)</center>

在正式安装VMT之前，首先链接您的VR设备并且启动SteamVR。

<center>![install_vmt_3](image/install_vmt/3.jpg)</center>

运行vmt_manager.exe

<center>![install_vmt_4](image/install_vmt/4.jpg)</center>

点击Install按钮，进行安装。随后，SteamVR需要重新启动以启用新增的加载项。

<center>![install_vmt_5](image/install_vmt/5.jpg)</center>

<center>![install_vmt_6](image/install_vmt/6.jpg)</center>

<center>![install_vmt_7](image/install_vmt/7.jpg)</center>

运行我们的MakiFBT软件，在SteamVR的设备界面应该可以看到新增的追踪器，尽管他们尚未正常工作。

<center>![install_vmt_8](image/install_vmt/8.jpg)</center>

重新打开vmt_manager.exe，切换到RoomSetup选项卡，输入您的RoomMatrix并点击SetRoomMatrix。如果您不清楚什么是RoomMatrix，直接点击ResetRoomMatrix。

<center>![install_vmt_9](image/install_vmt/9.jpg)</center>

<center>![install_vmt_10](image/install_vmt/10.jpg)</center>

至此，所有的追踪器应该正常运行。

<center>![install_vmt_11](image/install_vmt/11.jpg)</center>
