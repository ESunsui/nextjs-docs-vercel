# Parameter Modification

在MakiFBT中，我们定义了一系列参数来完成真实空间与游戏世界的映射关系。只有合理的参数才能使你获得正确的动作捕捉结果。

In MakiFBT, we define a series of parameters to complete the mapping relationship between the real space and the game world. Only reasonable parameters can enable you to obtain correct motion capture results.

在MakiFBT中，这一系列参数可以依据您的状态自动的生成，这一系列参数在大多数时候都是正确的。

In MakiFBT, this series of parameters can be automatically generated according to your state, and this series of parameters is correct most of the time.

然而，如果参数出现了问题，或是您想要做一些微调，您可以直接对最主要的几个参数直接做出修改。

However, if there is a problem with the parameters, or if you want to do some fine-tuning, you can directly modify the most important parameters.

要手动超控参数，请勾选“Enable Parameter Modification”选项框。

To manually override a parameter, check the Enable Parameter Modification option box.
